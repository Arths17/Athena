# Examples – Learning Athera

This folder contains 8 example programs demonstrating Athera's features.

## Quick Start

```bash
# Run any example
athera run examples/hello_world.ath
athera run examples/variables.ath
athera run examples/tasks.ath
# ... and so on
```

## Examples Included

| File | Topic | Complexity |
|------|-------|------------|
| `hello_world.ath` | First program | Beginner |
| `variables.ath` | Variables & lists | Beginner |
| `tasks.ath` | Functions/parameters | Beginner+ |
| `file_ops.ath` | File I/O | Intermediate |
| `text_processing.ath` | String operations | Intermediate |
| `math_ops.ath` | Arithmetic | Intermediate |
| `conditions.ath` | Loops & conditionals | Intermediate |
| `multimodule.ath` | Using multiple stdlib modules | Advanced |

## Learning Path

1. **Start here**: `hello_world.ath` – Verify installation
2. **Then**: `variables.ath` – Learn data storage
3. **Next**: `tasks.ath` – Write reusable functions
4. **Then**: `file_ops.ath` – Work with files
5. **Continue**: Other examples in any order
6. **Finally**: `multimodule.ath` – Put it all together

## Modifying Examples

Feel free to edit these files and experiment:

```bash
# Edit an example
nano examples/hello_world.ath

# Make changes, save, run:
athera run examples/hello_world.ath
```

## Creating Your Own

Use these examples as templates for your own programs:

```bash
# Copy and modify an example
cp examples/hello_world.ath my_program.ath
# Edit my_program.ath and run it
athera run my_program.ath
```

## Sharing

Athera programs are single files. Share them easily:

```bash
# Someone with athera installed can run:
athera run your_program.ath
```

No setup needed, no dependencies—just code!
